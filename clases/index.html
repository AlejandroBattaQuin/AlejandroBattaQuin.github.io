Sintaxis básica de JavaScript

Variables: Se utilizan para almacenar datos. Se declaran con la palabra clave var, let o const.

                                                
                                                    var nombre = "Juan";
                                                    let edad = 30;
                                                    const PI = 3.14;

Tipos de datos: JavaScript es un lenguaje de tipado dinámico, lo que significa que no es necesario declarar explícitamente el tipo de dato de una variable.

                                                
                                                var numero = 10;          // Número entero
                                                var decimal = 3.14;       // Número decimal
                                                var texto = "Hola";       // Cadena de texto
                                                var booleano = true;      // Valor booleano (verdadero/falso)
                                                var array = [1, 2, 3];    // Arreglo
                                                var objeto = {            // Objeto
                                                    nombre: "Juan",
                                                    edad: 30
                                                };

Operadores: Se utilizan para realizar operaciones matemáticas, comparaciones y lógicas.

                                                
                                                var suma = 10 + 5;
                                                var resta = 10 - 5;
                                                var multiplicacion = 10 * 5;
                                                var division = 10 / 5;
                                                var igualdad = (10 === 5);       // Igualdad estricta
                                                var mayorQue = (10 > 5);         // Mayor que
                                                var andLogico = (true && false); // Operador AND lógico

Funciones: Se utilizan para agrupar y reutilizar código. 

                                                
                                                function saludar(nombre) {
                                                    return "Hola, " + nombre + "!";
                                                }
                                                
                                                var mensaje = saludar("Juan");
                                                console.log(mensaje); // Imprime: Hola, Juan!

Condicionales: Se utilizan para tomar decisiones basadas en condiciones. 

                                                
                                                var edad = 18;
        
                                                if (edad >= 18) {
                                                    console.log("Eres mayor de edad");
                                                } else {
                                                    console.log("Eres menor de edad");
                                                }

Bucles: Se utilizan para repetir una porción de código varias veces.

                                                
                                                for (var i = 0; i < 5; i++) {
                                                    console.log("Iteración número: " + i);
                                                }
                                                
                                                var i = 0;
                                                while (i < 5) {
                                                    console.log("Iteración número: " + i);
                                                    i++;
                                                }
